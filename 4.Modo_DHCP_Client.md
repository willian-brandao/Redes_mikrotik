
# Configurar um roteador no modo DHCP Client

## Desabilitar a interface de PPPoE do dispositivo acessando PPP > Interface 

   Para desativar a interface, clica-se no "X" logo acima da listagem de interfaces.

## Habilitar a interface física que leva link até o roteador

  Para ativar, acessa a sessão de IP > DHCP Client. 

  <img width="886" height="389" alt="image" src="https://github.com/user-attachments/assets/c91e63d0-ea47-440d-9ccb-adc44958823b" />

  Acessando o "+" para criação da interface, seleciona o interface ethernet desejada no campo interface e configura como "yes' o campo "Add Default Route", depois "apply" e "ok".

  <img width="546" height="511" alt="image" src="https://github.com/user-attachments/assets/efe92138-84be-4186-bd23-e3024a9ca4da" />
